# Restaurant Website

This project is for a high-end restaurant, focusing on a modern interface and a smooth user experience.

## 🔗 Live Demo
https://thuanngo3008.github.io/Restaurant_Project/src/
## 🛠️ Technology used
* **Frontend:** HTML5, CSS3, JavaScript/jQuery.
* **Responsive:** Displays well on all devices (Mobile, Tablet, Desktop).
* **Deployment:** GitHub Pages.

## ✨ Features
* Professional food menu interface.
* The visual effects and page scrolling are smooth.
* Integrated contact and reservation form.
